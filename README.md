# Excel-Dashboard
## This project features a data visualization dashboard with six charts: a stacked bar chart for smart city technologies, a bar chart showing energy savings using technology, a monthly revenue area chart, a regional profit pie chart, a line chart for key business metrics, and a donut chart showing subject-wise marks distribution. 📊
## This project showcases an interactive Excel Dashboard created to visualize and analyze data effectively. It converts raw data into meaningful charts and insights using Microsoft Excel.
## Tools Used
## Microsoft Excel
## Charts & Graphs
## Pivot Tables
## How to Use
## Download the Excel file and open it in Microsoft Excel to explore the dashboard and data insights.
